## Tech Stack

| Layer | Technology | Notes |
|-------|-----------|-------|
| Platform | Shopify | Chosen for mature subscription-app ecosystem |
| Theme | **Decided: Vessel** (paid Shopify Theme Store theme) | User's choice; installed/connected directly by user via Shopify Admin. Proprietary — not vendorable into this repo like Dawn was. Superseded the earlier Dawn vendor (removed from repo). |
| Subscriptions | Recommended: Skio | Native Shopify checkout/customer-account integration, lower cost at low volume vs. Recharge. Pending user confirmation. |
| Payments | **Decided: Shopify Payments** | Square dropped for this site. Shopify Payments = 2.9%+30¢ online on Basic plan (matches Square, no added surcharge since it's Shopify's native gateway), dropping to 2.6%/2.4%+30¢ on higher plans. Works natively with Skio/Recharge recurring billing. |

## Architecture

- Theme: Vessel, installed and connected by the user directly in Shopify Admin (Theme Store purchase + activation) — not managed through this repo.
- Any customization beyond what Vessel's theme editor/customizer supports (e.g. a custom plan-builder section) requires a Theme Access token so Claude can pull/push its Liquid code via Shopify CLI — not yet set up; only needed if the native customizer can't cover the plan-builder UX.
- Subscription logic (recurring billing, customer portal) lives in the subscription app (Skio, pending confirmation), integrated into the theme via app blocks/sections — not built custom in Liquid.
- Store data/catalog changes (products, collections, pricing) go through the Shopify MCP connector, independent of theme choice.

## Environment & Setup

- Shopify store connected via Shopify MCP connector: **Brown Bag Coffee Club** (`brown-bag-coffee-club.myshopify.com`), Basic plan, USD, US/EDT
- MCP connector gives Claude scoped store access (products, collections, orders, customers, inventory, analytics, discounts, and general Admin GraphQL) — no raw admin credentials exchanged
- Theme (Vessel) is installed/managed by the user directly in Admin, not through this repo. If deeper Liquid customization is needed later, a Theme Access token would let Claude pull/push Vessel's code via Shopify CLI.

## Known Constraints / Gotchas

- Claude Code can write/customize theme code but cannot perform Shopify admin actions directly (store creation, app installation, payment/tax setup) — those require the user to act in the Shopify dashboard, or to grant API/CLI access.
- Reference site's standard (non-promotional) pricing and full FAQ content are JS-rendered and weren't retrievable via WebFetch's HTML summarizer — worked around by pulling the raw catalog from Shopify's public `/products.json` endpoint instead (see Product Catalog below), which returns real prices directly.
- Atlas's live store contains many near-duplicate products (ad-campaign-specific variants, e.g. tagged `facebook-2023`, plus a straight "10% off" line) with different prices for what's otherwise the same product. The pricing table below uses the variant that appeared identically across multiple non-discounted listings, treated as their standard baseline — not a guarantee it's their exact current sticker price.
- Atlas does not sell specific named single-origin coffees as separate SKUs — the origin rotates monthly and is communicated via the postcard/tasting notes insert, not as a selectable product option. The purchasable variant axis is **Roast Profile** (Smooth & Balanced / Bright & Expressive), not origin. Our v1 catalog mirrors this structure; origin copy is placeholder/rotating content, not a product field.

## Reference Site Notes — atlascoffeeclub.com

- **Nav**: Our Coffee, Subscription, Gifts, Account Login, Cart
- **Subscription collection** filtered by brew method via query param: `?brew=drip_machine`, `pods`, `french_press`, `espresso`; decaf is a separate collection (`/collections/subs-decaf`)
- **Box contents**: 12 oz bag, tasting notes, postcard from origin country, coffee history
- **Gift product**: single product page with curated bundle variants (coffee + chocolate, coffee + tea pairing), gift note (max 6 lines / 80 chars per line), mail or email delivery, email delivery scheduled for 8:00am CST
- **Flexibility messaging**: "Pause, skip, or cancel anytime"; "30-Day Money Back Guarantee"; free U.S. shipping
- **Community**: separate subdomain (`club.atlascoffeeclub.com`) — likely forum/rewards, treated as out of scope for v1 (see Project Overview)
- **Footer/legal pages**: About Us, Media, Brew Guides, FAQ, Contact Us, Terms & Conditions, Privacy Policy, Cookie Policy, CCPA request page, Accessibility Statement
- **Social proof**: press mentions (New York Magazine, BuzzFeed, Eater, Boston Globe)
